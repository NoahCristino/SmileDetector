SmileDetector - UTMIST

Smile Detector detects smiles in images using a new multi-step approach which consists of first performing facial recognition and then running the detected facial vectors though our custom C-Support Vector Classification model. By reducing the model input size and approaching this problem in a new way, we were able to produce a model that can perform classifications with high accuracy in fractions of the time that traditional models take. There are many applications for this technology including engagement analysis, marketing, and photography.

  • Realtime Emotion Detection

  • Fastest Existing Model

    Our model performs detections in fractions of the time that other similar models take which gives you more resources to perform additional analysis.

  • Lightweight

    Our model uses a new approach to smile detection which is far more lightweight than competing models. We process facial vectors extracted from the given image rather than the entire image which saves on memory and allows us to outperfom competing models.

  • Facial Landmark Detection

    Facial Detection is done based off of facial landmarks which allows us to isolate key facial features from a given image in a millisecond.

  • ~93% Accuracy

    Our model was trained on thousands of images and when tested on thousands of frames of test data it performed with ~93% accuracy. Please see the medium article for further explanation.

  • Runs in Browser

    No installation required! The model is fast enough to process webcam footage streamed from the browser. Everything has been ported to the web and is hosted by Heroku.
